Congratulations! You've successfully installed the Enterprise Edition of Kaazing WebSocket Gateway ${gateway.version}

This is the default index.html page located in:

$GATEWAY_HOME/web/welcome

and the configuration file is:

$GATEWAY_HOME/conf/gateway-config.xml

where $GATEWAY_HOME is the root of your gateway installation.

Want to test a WebSocket connection to this gateway?

  1. Open the websocket.org echo test page

  2. Enter in the Location field

  3. Connect and send a message
